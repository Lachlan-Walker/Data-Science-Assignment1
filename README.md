The objective of this assignment is to extract a dataset from a set of web pages and
use Python to prepare, analyse, and derive insights from the collected data.
Choose one of the four data sources listed here:
http://mlg.ucd.ie/modules/python/sources 
The assignment should be implemented as two Jupyter notebooks (not script files).
The notebooks should be clearly documented, using comments and Markdown cells to
explain the steps applied in the code and interpret the results of the analysis.
